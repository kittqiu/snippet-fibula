# snippet-fibula
a nodejs powered, light weight snippet manage.


## Environment

Nodejs: v4.2.2


## Install packages
	* install codash
		$ npm install codash

	* Run `npm install` to install all required packages.
		$ npm install
		$ npm install gulp -g

	* make static files by gulp
		$ gulp
		$ gulp babel
		$ gulp uglify
